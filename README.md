How to Use VoteSure

This guide explains how a new admin and a new voter use the system.

Admin Guide
1. Log in
   - Open the admin login page and sign in.
2. Create an election
   - Go to `Elections`.
   - Add a new election and set the title, dates, and description.
3. Add party lists
   - Go to `Partylist`.
   - Create party lists for the election.
4. Add candidates
   - From the party list table, click `Candidates`.
   - Add a candidate for each position (photo optional).
5. Start or stop voting
   - Go to `Start / Stop`.
   - Set the election status to `running` to open voting.
6. Monitor results
   - Go to `Results` to see live counts.
   - Use `Reports` to export votes if needed.
7. Manage voters
   - Go to `Registered Voters` to edit or remove voter records.

Voter Guide
1. Open the voter page.
2. Enter your Student ID.
   - Must be exactly 9 digits (numbers only).
3. If prompted, complete your profile.
   - Full name, strand, and year level.
4. Vote
   - Select one candidate per position.
   - Review and submit your vote.
5. Confirmation
   - You will see a confirmation screen after submitting.

Rules and Notes
- Voting is only available while an election is set to `running`.
- Each student can vote once per election.
- Student IDs are auto-registered during an active election.
